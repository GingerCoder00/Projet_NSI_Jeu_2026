Faire les jauges suivantes :
    1️⃣ 🌡 Température globale

    Monte avec les émissions.
    Impacte presque toutes les autres jauges.
    Seuils critiques → événements extrêmes.

    2️⃣ 🏭 Pollution / CO₂

    Jauge “moteur” principale.
    Augmente via industrie, transports, guerres…
    Fait monter la température.

    3️⃣ 🌊 Niveau des océans

    Monte avec la température.
    Réduit la population et la stabilité.

    4️⃣ 🌳 Biodiversité

    Baisse avec pollution + température.
    Si trop basse → effondrement alimentaire.

    5️⃣ 🌾 Ressources alimentaires

    Dépend de l’eau, biodiversité, climat.
    Si faible → famine → instabilité.

    6️⃣ 👥 Stabilité mondiale

    Impactée par famine, catastrophes, migrations.
    Si très basse → guerres (qui augmentent pollution 👀).
Faire la jauge de progression total : 

    📊 Jauge 7 : ☠ Progression de destruction globale

    Elle pourrait être calculée par exemple comme :
    
    Destruction = 
    (Température * 0.2) +
    (Pollution * 0.2) +
    (Niveau océans * 0.15) +
    ((100 - Biodiversité) * 0.15) +
    ((100 - Ressources alimentaires) * 0.15) +
    ((100 - Stabilité) * 0.15)

    👉 Plus elle monte, plus on se rapproche de l’extinction / collapse final.
Coder les interactions entre jauges :

    🎮 Structure générale du modèle

    On suppose que toutes les jauges vont de 0 à 100.

    temperature      # 0 → 100
    pollution        # 0 → 100
    oceans           # 0 → 100
    biodiversite     # 0 → 100
    nourriture       # 0 → 100
    stabilite        # 0 → 100

    🧠 Logique des interactions

    Chaque "tour" (ou chaque seconde) tu appelles :

    def update_world(dt):
    dt = temps écoulé (ex: 1 seconde)

    🔥 1️⃣ Pollution → Température

    Relation progressive mais ralentie :

    temperature += pollution * 0.002 * dt


    👉 Plus la pollution est élevée, plus la température augmente vite.

    🌊 2️⃣ Température → Océans
    oceans += temperature * 0.0015 * dt

    🌳 3️⃣ Pollution + Température → Biodiversité
    biodiversite -= (pollution * 0.001 + temperature * 0.0015) * dt

    🌾 4️⃣ Biodiversité + Température → Nourriture
    nourriture -= ((100 - biodiversite) * 0.001 + temperature * 0.001) * dt

    👥 5️⃣ Nourriture + Océans → Stabilité
    stabilite -= ((100 - nourriture) * 0.002 + oceans * 0.001) * dt

    💥 6️⃣ Stabilité basse → Pollution augmente (guerres)

    Effet boule de neige :

    if stabilite < 40:
            pollution += (40 - stabilite) * 0.01 * dt

    📊 Jauge globale de destruction
    destruction = (
            temperature * 0.2 +
            pollution * 0.2 +
            oceans * 0.15 +
            (100 - biodiversite) * 0.15 +
            (100 - nourriture) * 0.15 +
            (100 - stabilite) * 0.15
    )

🔥 3️⃣ Système de propagation dynamique (très stylé)

	Quand tu fais :
	self.ajout_feu(ligne, colonne)

	Ajoute une probabilité de propagation :

	def propagation_feu(self, ligne, colonne):
    		directions = [(1,0), (-1,0), (0,1), (0,-1)]
    
    		for dx, dy in directions:
        		new_l = ligne + dx
        		new_c = colonne + dy
        
        		if 0 <= new_l < self.lignes and 0 <= new_c < self.colonnes:
            		if randint(0,100) < 30:  # 30% chance
                		self.ajout_feu(new_l, new_c)

	🔥 Et si température élevée → 60% au lieu de 30%.
	Ça crée un vrai chaos progressif.
