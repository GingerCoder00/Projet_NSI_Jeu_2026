# Sitothèque — Ressources pour le développement de jeu

> Toutes ces ressources sont **gratuites** et accessibles **sans création de compte**.

---

## Images & Sprites

### [PNGWing](https://www.pngwing.com/)

| Champ | Détail |
|-------|--------|
| Accès | Libre, sans compte |
| Description | Bibliothèque d'images PNG avec fond transparent (images découpées / sans fond) et sprites prêts à l'emploi. Idéal pour intégrer des visuels directement dans un projet sans retouche. |

---

### [Piskel App](https://www.piskelapp.com/)

| Champ | Détail |
|-------|--------|
| Accès | Libre, sans compte |
| Description | Outil en ligne de création de sprites animés et de backgrounds pixel art. Permet d'exporter en PNG, GIF ou spritesheet. Parfait pour concevoir les assets graphiques d'un jeu. |

---

### [Pixeliser.fr](https://pixeliser.fr/)

| Champ | Détail |
|-------|--------|
| Accès | Libre |
| Description | Outil permettant de pixeliser une image. Utilisé notamment pour créer des fonds d'écran stylisés pour le menu du jeu. |

---

## Visualisation de données

### [Plotly — Gauge Charts](https://plotly.com/python/gauge-charts/)

| Champ | Détail |
|-------|--------|
| Accès | Libre, sans compte |
| Description | Documentation officielle Plotly Python pour la création de jauges (gauge charts). Fournit un code Python clé en main et un aperçu visuel du rendu. Utile pour afficher des indicateurs (vie, énergie, score...). |

---

## Audio & Musique

### [SFXR](https://sfxr.me/)

| Champ | Détail |
|-------|--------|
| Accès | Libre, sans compte |
| Description | Générateur de bruitages rétro en ligne. Permet de créer des effets sonores style 8-bit (tirs, sauts, explosions, power-ups...) et de les exporter en fichier audio. |

---

### [BeepBox](https://www.beepbox.co/)

| Champ | Détail |
|-------|--------|
| Accès | Libre, sans compte |
| Description | Compositeur de musiques rétro en ligne. Interface intuitive pour créer des mélodies et boucles musicales au style chiptune / 8-bit, exportables en fichier audio. |
