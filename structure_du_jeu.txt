CONCEPT DU JEU

Jeu pixel art satirique arcade (~10 minutes).

Tu incarnes une méga-corporation qui accélère volontairement l’effondrement climatique.

Objectif : remplir la Jauge de destruction globale le plus vite possible.

LES JAUGES PRINCIPALES (0 → 100)
Pollution

Moteur principal de destruction.

Augmente avec les pouvoirs industriels

Fait monter la température

Fait baisser biodiversité

Température

Dépend directement de la pollution.

Monte automatiquement avec pollution

Fait baisser eau disponible

Fait baisser biodiversité

Eau disponible

Représente les ressources en eau potable.

Baisse avec température

Si trop basse → stabilité chute

Biodiversité

Représente écosystèmes / faune / flore.

Baisse avec pollution + température

Si trop basse → stabilité chute plus vite

Stabilité sociale

Représente paix mondiale / gouvernements.

Baisse si eau ou biodiversité faibles

Si très basse → augmente pollution (guerres, chaos)

Effet boule de neige.

Profit / Coût

Jauge stratégique.

Sert à activer les pouvoirs

Augmente via industrialisation

Plus le profit est haut → pollution augmente automatiquement

Exemple :

pollution += profit * 0.04 * dt

Plus tu gagnes d’argent, plus tu détruis vite.

Jauge de destruction globale

Elle dépend de toutes les autres.

Formule simple :

destruction = (
    pollution * 0.25 +
    temperature * 0.2 +
    (100 - eau) * 0.15 +
    (100 - biodiversite) * 0.15 +
    (100 - stabilite) * 0.15 +
    profit * 0.1
)


Quand destruction ≥ 100 → victoire.

SYSTÈME D’ÉVOLUTION AUTOMATIQUE (update_world)

Chaque frame / seconde :

temperature += pollution * 0.002 * dt

eau -= temperature * 0.0015 * dt

biodiversite -= (pollution * 0.001 + temperature * 0.001) * dt

if eau < 40:
    stabilite -= (40 - eau) * 0.01 * dt

if biodiversite < 40:
    stabilite -= (40 - biodiversite) * 0.008 * dt

if stabilite < 30:
    pollution += (30 - stabilite) * 0.02 * dt

pollution += profit * 0.001 * dt


Ça crée une réaction en chaîne progressive.
